## Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd lab8
Запустить контейнеры:
```bash
docker-compose up -d --build
```
Открыть в браузере:
```http://localhost:8080```

Содержимое проекта:\

```tests/``` — папка с тестами PHPUnit (unit-тесты, mock-тесты, HTTP-тесты)

```tests/ExampleTest.php``` — базовый тест для проверки работы PHPUnit

```tests/ConferenceTest.php``` — тесты для класса Conference (добавление и получение данных, mock PDO)

```tests/ApiTest.php``` — тестирование HTTP-запросов через Guzzle (с использованием MockHandler)

```www/Conference.php``` — основной класс приложения

```www/index.php``` — точка входа в приложение (используется для HTTP-запросов и тестирования API)

```composer.json``` — файл зависимостей проекта (Guzzle, PHPUnit)

```vendor/``` — папка с установленными библиотеками Composer

```composer.lock``` — файл фиксации версий зависимостей проекта

```.env.test``` — файл конфигурации тестовой среды
